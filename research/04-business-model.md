# Modelo de Negocio y Recuperación de Inversión — EcoRuta

## 1. Modelo de ingresos: SaaS por suscripción

### Estructura de pricing

| Plan | Descripción | Precio estimado (MXN/mes) | Incluye |
|------|-------------|--------------------------|---------|
| **Piloto** | 1 zona, hasta 50 contenedores | Gratis (3-6 meses) | Dashboard básico, optimización de rutas, reportes |
| **Básico** | 1 alcaldía, hasta 200 contenedores | $25,000 - $40,000 | Todo lo anterior + IA generativa, app conductor, reportes ejecutivos |
| **Pro** | Múltiples zonas, hasta 500 contenedores | $60,000 - $100,000 | Todo lo anterior + API de integración, predicciones ML, soporte prioritario |
| **Enterprise** | Ciudad completa, ilimitado | Cotización a medida | Todo lo anterior + personalización, SLA, consultoría |

**Referencia de mercado:** El software de smart waste management captura 41% del mercado global y crece al 20% anual con modelos SaaS.

## 4. Modelo de go-to-market

### Fase 1: Proof of Concept (0-6 meses post-hackathon)
- **Estrategia:** Piloto gratuito con 1 alcaldía de CDMX.
- **Objetivo:** Generar datos reales de ahorro para usar como caso de éxito.
- **Costo para EcoRuta:** ~$50,000 MXN (infraestructura cloud + tiempo del equipo).
- **Ingresos:** $0 (inversión).

### Fase 2: Primeros clientes (6-18 meses)
- **Estrategia:** Convertir piloto en contrato pagado + expandir a 2-4 alcaldías con caso de éxito.
- **Objetivo:** 3-5 clientes pagando suscripción Básico/Pro.
- **Ingresos estimados:** $1.5M - $3.6M MXN/año.

### Fase 3: Escalamiento (18-36 meses)
- **Estrategia:** Licitación con gobierno central CDMX + expansión a GDL, MTY, Puebla.
- **Objetivo:** 10-15 clientes, mix gobierno + privado.
- **Ingresos estimados:** $7M - $15M MXN/año.

### Fase 4: Regional (3-5 años)
- **Estrategia:** Expansión a otras ciudades de LATAM (Bogotá, Lima, Santiago).
- **Ingresos estimados:** $30M+ MXN/año.
